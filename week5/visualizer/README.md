# Week5 Visualizer

DP 알고리즘 시각화 도구 모음

## TSP Visualizer (비트마스크 DP + DFS)

4개 도시 외판원 문제를 비트마스크 DP로 풀어가는 과정을 단계별로 시각화합니다.

- 그래프 위 현재 위치 및 경로 표시
- 비트마스크 상태 해석
- 재귀 호출 스택
- DP 테이블 실시간 갱신

### 실행 방법

```bash
cd week5/visualizer
npm install
npm run dev
```

http://localhost:5173 에서 확인

## LCS Visualizer

LCS(최장 공통 부분 수열) 알고리즘 시각화. `lcs_visualizer.html`을 브라우저에서 직접 열면 됩니다.
